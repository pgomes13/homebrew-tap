# Drift Guard Homebrew Tap

This is the official Homebrew tap for [Drift Guard](https://github.com/pgomes13/drift-guard-engine), a schema diff engine that detects breaking vs. non-breaking API contract changes across OpenAPI, GraphQL, and gRPC schemas.

## Installation

To install Drift Guard using Homebrew, first add this tap:

```bash
brew tap pgomes13/drift-guard
```

Then install Drift Guard:

```bash
brew install drift-guard
```

## Usage

After installation, you can run Drift Guard from the command line:

```bash
drift-guard --help
```

For more information, visit the [main repository](https://github.com/pgomes13/drift-guard-engine).
